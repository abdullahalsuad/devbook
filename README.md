# Report an Issue

If you find any **errors, mistakes, or missing information**, please feel free to create an issue. Follow these steps:

1. Go to the [Issues page](https://github.com/abdullahalsuad/devbook/issues) of this repository.
2. Click on **New Issue**.
3. Write a **clear title** describing the problem.
4. Explain the issue **step by step** (example: what you tried, what happened, expected behavior).
5. Add any **screenshots or code snippets** if needed.
6. Click **Submit new issue**.
